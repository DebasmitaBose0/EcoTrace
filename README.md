---

## 🚀 Key Features

* ⏱️ **Parallel & Efficient Page Loader**: A sleek, dark-mode splash screen featuring animated progress bars, dynamic status indicators, and changing ecological tips. Connects to the backend configuration endpoints in parallel to optimize initial load speeds.
* 📈 **Diagrammatic X-Y Axis Trend Line**: A custom-built vector SVG graph with projection dashed guidelines connecting actual daily logs against your custom baseline targets over time.
* 💾 **Exportable Reports**: One-click **Download PDF Report** formatting using media print styling to clean up and layout the stats for print or saving, and **Export CSV Data** downloading for offline spreadsheet audits.
* 🤖 **Secure AI Eco-Coach Integration**: Secure backend proxy communicating with the state-of-the-art `gemini-3.5-flash` model utilizing system instructions customized to three distinct focus personas (**Aero**, **Sprout**, or **Kelvin**).
* 📝 **Daily Green Actions Log**: Track your daily vehicle commutes, transit times, dietary portions, home energy actions (thermostat offsets, vampire power cuts, shorter showers), and waste redirection.
* 🏆 **Habit Commitments & Pledges**: Commit to tailored pledges generated by AI to maximize weekly carbon savings.

---

## 🛠️ Adaptive Focus Personas

The application features an **Adaptive Focus Persona framework** that maps directly to the verticals:

1. **🚴 Aero (Transit & Mobility Coach)**: Focuses on vehicle parameters, EV conversions, carpooling, micromobility, direct flight offsets, and transit optimizations.
2. **🥗 Sprout (Conscious Food & Agri Coach)**: Focuses on plant-forward transitions, local organic sourcing, food waste minimization, and kitchen energy savings.
3. **🏠 Kelvin (Home Resource Consultant)**: Focuses on passive heating/cooling offsets, weatherization, vampire power savings, hot water conservation, and solar power.

## 📐 Scientific Calculation Factors (kg CO2e)

All emission calculations are mathematically modeled based on recognized greenhouse gas protocols:
- 🚗 **Vehicles (per mile)**: Gas SUV (`0.46`), Gas Sedan (`0.32`), Hybrid (`0.18`), Electric (`0.08`).
- 🚌 **Public Transit**: `0.08 kg` per mile.
- 🥩 **Dietary Profiles (daily)**: Heavy Red Meat (`7.5`), Standard (`5.4`), Poultry/Egg (`3.9`), Vegetarian (`2.6`), Vegan (`1.7`).
- ⚡ **Electricity (per kWh)**: Traditional fuel utility grid (`0.42`), mixed/solar tariffs (`0.22`), solar (`0.04`).
- 🗑️ **Waste Management**: Composting and recycling yield offsets of `-0.4 kg` and `-0.7 kg` daily.

## 📦 Tech Stack

- **Core**: React 19, TypeScript 5, Vite 6, Express 4.
- **Styling**: Vanilla CSS with Tailwind CSS v4.
- **Icons**: Lucide React.
- **AI Engine**: Google GenAI SDK (`gemini-3.5-flash`).

## ⚙️ Getting Started

### 1. Configure Secret API Key
Create a `.env` file in the root directory (already ignored by Git in `.gitignore`):
```env
GEMINI_API_KEY="your-google-ai-studio-gemini-key"
APP_URL="http://localhost:3000"
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Pipeline
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view your local deployment.

### 4. Production Compilation & Build
```bash
npm run build
npm start
```
